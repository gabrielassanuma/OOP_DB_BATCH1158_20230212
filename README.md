# OOP_DB_batch1158_20230212

## Run Locally

Clone the project

```bash
  git clone https://github.com/gabrielassanuma/OOP_DB_batch1158_20230212
```

Go to the project directory

```bash
  cd OOP_DB_batch1158_20230212/
```

Open on vs-code

```bash
  code .
```

Create DB

```bash
  rake db:create
  rake db:migrate
  rake db:seed
```

Make sure you are on the right folder.
